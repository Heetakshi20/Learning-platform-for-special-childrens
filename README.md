# 🌟 BrightMinds – AI Career Guidance System

## 📌 Overview
BrightMinds is an AI-powered career guidance platform designed to help students and individuals discover personalized career paths. It analyzes user interests, skills, and goals to provide intelligent recommendations and actionable insights.

---

## 🚀 Features
- 🎯 Personalized career recommendations  
- 📊 Skill and interest analysis  
- 🤖 AI-driven decision support  
- 📚 Suggested learning resources  
- 📈 Career growth roadmap  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript / React  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Analytics:** Python, SQL
